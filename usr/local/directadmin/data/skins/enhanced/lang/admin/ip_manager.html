LANG_LINK_OPTIONS_EXPLANATION=Het toevoegen van een gekoppeld IP adres aan het huidige IP adres zal de gekoppelde IP adres toevoegen aan het gebied waar het momentele IP adres bestaat.
Koppelen van een IP adres met DNS zal de A/AAAA records dupliceren voor het huidige IP adres met het gekoppelde IP adres.
Koppelen van een IP adres met Apache zal het gekoppelde IP adres toevoegen aan de VirtualHost met het huidige IP adres. LANG_MULTI_IP_RELATION=Deze feature is afhankelijk van het multi-IP systeem, maar kunnen tegelijkertijd gebruikt worden.
Zo kan bijvoorbeeld een gebruikt 2 IP adressen hebben met het multi IP systeem, en elk IP adres kan zijn eigen gekoppelde IP adressen hebben. LANG_LINK_NOT_RECURSIVE=Links zijn niet recursief, er zal dus geen oneindige lus ontstaan. LANG_APPLY_TO_DOMAINS=De "Pas toe op bestaande domeinen" optie zal de http.config rewrites triggeren, en A/AAAA records toevoegen aan de huidige lokale DNS zones.
Het uitschakelen van deze checkbox zal ervoor zorgen dat de http.conf bestanden en DNS instellingen niet worden aangepast, maar toekomstige httpd.conf rewrites zullen het gekoppelde IP adres bevatten.
